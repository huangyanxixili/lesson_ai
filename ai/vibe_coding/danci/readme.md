## Vibe coding
- 给自己做个小工具产品
- Vibe Coding 是一种自我表达方式，简历中最重要的

## 提示词
- 清晰的给出指令
- 分步做
- 有规划
- 不要做什么
- 界面，风格提出需求

## gemini 的能力
- 为什么能优质的生成整个中小型应用？
   - 代码能力 
   - gemini支持更大的上下文窗口（接受复杂的提示词）
        单次 token[词元] 的长度（可到达100万tokens）
   - llm 就像一个简单的函数：
        prompt = 输入
        output = 输出

- 如果没有经历产品需求、原型设计，vibe coding 提示词就不可能专业，所以vibe coding 是一种自然语言编程，传统敲代码之外的编程方式，vibe coding 也有它的开发流程和工艺
