## 三数之和

- 两数之和 --> 暴力： O(n^2)
    hashMap 变成求差，判断差是否在 hashMap 中
- 三数之和 --> 暴力： O(n^3) 
    先用”快排“来排序 O(nlogn) -> 从小到大
      - 方便跳过重复的数字
      - 固定一个数（前面一定小，后面一定大）
      - 双指针 来求另外两个数
          - 左指针 从 固定数的下一个 开始
          - 右指针 从 数组最后 开始
          - 固定数 + 左指针 + 右指针 = 0 ，加入结果集
          - 固定数 + 左指针 + 右指针 < 0 ，左指针右移
          - 固定数 + 左指针 + 右指针 > 0 ，右指针左移
