# 删除链表节点

## dummy 节点
- 节点迭代
- 删除节点
- dummy 节点（哨兵节点）虚拟的节点
    是一个人为添加的节点，不存储真实数据的假节点，通常放在列表的最前面（有时候在尾部），目的是简化链表操作的边界条件
    用空间换逻辑统一

## 链表的反转
    通过 dummy 节点 + 头插法
    dummy 充当了 反转后的头节点
    想清楚每轮的关系
  - 头插法分为三部：
    1. 保存下一节点             `const next = cur.next`
    2. 让当前节点指向已经反转的头 `cur.next = dummy.next`
    3. 成为新的反转头           `dummy.next = cur`
